

## OAuth Bypasses
- **Improper redirect_uri validation** → **Authentication bypass**
- **Authorization code reuse** → **Authentication bypass**
- **Authorization code not bound to client_id** → **Authentication bypass**
- **State parameter missing or not validated** → **Authentication bypass (CSRF)**
- **Implicit flow token leakage (fragment/referrer)** → **Authentication bypass**
- **OpenID Connect `id_token` not validated (iss/aud/signature)** → **Authentication bypass**
- **Using access token as proof of authentication** → **Authentication bypass**
- **PKCE missing or not enforced (public clients)** → **Authentication bypass**
- **Account linking without ownership verification** → **Authentication bypass**
