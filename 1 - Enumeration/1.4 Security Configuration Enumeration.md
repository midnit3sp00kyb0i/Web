# X-Frame-Options / Frame-Ancestors
```
curl -sI https://target.xyz | grep -iE 'x-frame-options|content-security-policy'
```
Bad values:
- No header
- `frame-ancestors *`
# X-Content-Type-Options
```
curl -sI $TARGET | grep -i x-content-type-options
curl -sI $TARGET/some.js | grep -i content-type
```
# Referrer Policy
```
curl -sI $TARGET | grep -i referrer-policy
curl -s $TARGET | grep -i '<meta.*referrer'
```
Good values:
- `no-referrer`
- `strict-origin`
- `strict-origin-when-cross-origin`
